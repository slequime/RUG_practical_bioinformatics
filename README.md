# Practical Bioinformatics for Biologists (University of Groningen)
![](https://img.shields.io/badge/-Teaching%20material-red?style=flat&logo=R)

This repository contains teaching material for the lecture on R in the course Practical Bioinformatics for Biologists, offered at the University of Groningen (The Netherlands). It provides an example for the student to use, as well as the solutions to the exercices (html and .Rmd) file.
